# facil-consulta-teste

## Faça um clone do projeto e execute na pasta o comando:
```
yarn install
```

### Inicie o projeto com: 
```
yarn serve
```

### O que foi utilizado?
```
- v-mask
- bootstrap-vue
- vue-router
```
### O que faltou?

- Responsividade com precisão
- Armazenamento e validação do formulário
